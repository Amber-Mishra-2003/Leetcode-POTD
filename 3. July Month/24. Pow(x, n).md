# Pow(x, n)

<h3>
Implement pow(x, n), which calculates x raised to the power n (i.e., xn).
</h3>

[Problem Link](https://leetcode.com/problems/powx-n/description/)

**Solution :-**

```
class Solution {
public:
    double myPow(double x, int n) {
        
        double  val ;

        val = pow(x,n);

       // cout<<val<<endl;

        return val;
    }
};
```
